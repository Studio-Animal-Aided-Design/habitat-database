# AAD Converter Dokumentation

Diese Dokumentation beschreibt den gesamten End-to-End-Prozess rund um den AAD Tooljet Converter, auch wenn die App nicht geoeffnet ist.

Verfuegbar:

- [AAD-Converter-Handbuch.md](</Users/ckonrad/Documents/Arise/Kunden/Studio Animal Aided Design/habitat-database/docs/converter/AAD-Converter-Handbuch.md>)
- `AAD-Converter-Handbuch.pdf` im selben Ordner

Zielgruppen:

- Datenliefernde, die Excel-Dateien vorbereiten
- Projektverantwortliche, die Konvertierungen durchfuehren
- Maintainer, die Builds erzeugen oder Probleme analysieren

Die PDF wird aus dem Markdown-Quelltext generiert:

```bash
python3 scripts/generate_converter_docs_pdf.py
```
